# Startly

A streamlined tool designed to accelerate frontend project setup

## Usage/Examples

```bash
startly
```

## Run Locally

#### Global installation

```bash
npm run build
npm i -g .
```

#### Local run

```bash
npm run startly
```

#### run in another project

```bash
cd path/to/your/project
startly
```

or

```bash
cd path/to/your/project
npx startly
```

## Requirements

System: Windows

Package menager: NPM
